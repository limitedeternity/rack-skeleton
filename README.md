
A Rack webserver skeleton, that I use to serve static webpages.

-----------

How to:
====================

*Login to Heroku:*

- "heroku login"

*Initialize git repo:*

- "git init"

*Bind git repo to your Heroku application:*

- "heroku git:remote -a <your_app>"

*Now do your stuff. When completed, deploy:*

- "ruby deploy.rb"
